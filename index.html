☁️ iCloud Drive Setup Guide
ℹ️
Your data file is always named OpticEMR-data.json. Keep it in one iCloud Drive folder and it stays in sync across your Mac and iPad.
Initial Setup (Mac)
  1. Open Finder → go to iCloud Drive
  2. Create a folder called OpticEMR
  3. Move ophtho-emr.html into that folder
  4. Open the app from that folder in Safari or Chrome
Saving (Mac & iPad)
  1. Click 💾 Save — downloads OpticEMR-data.json
  2. On Mac: move the file to your iCloud Drive → OpticEMR folder
    (replace the old file when prompted)
  3. On iPad: file saves to Files app → Downloads — move it to iCloud Drive → OpticEMR
Loading on iPad
  1. Open Files app → iCloud Drive → OpticEMR
  2. Tap ophtho-emr.html to open the app in Safari
  3. Tap 📂 Load → navigate to iCloud Drive → OpticEMR → OpticEMR-data.json
  4. Your records appear instantly
Add to iPad Home Screen (optional)
  1. Open the app in Safari on iPad
  2. Tap the Share button (↑ box icon)
  3. Tap "Add to Home Screen"
  4. It launches like a native app — tap Load each session to restore data
🔮 Upgrading to NAS later
When you get a NAS, just tell your developer (or Claude) — only the Save/Load functions need updating. All your patient data in the JSON file transfers directly with no changes needed.
